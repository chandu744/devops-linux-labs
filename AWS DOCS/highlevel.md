EC2(elastic cloud compute)
==========================
- ec2 instances==> vms or servers
- elastic machines->scaleup or down easily
- use nano, micro , small or medium for==> t1, t2,t3 instances
- use GP2(general purpose)volumes ==> WITH MAX STORAGE ==> 30GB
- max --> 3 stopped instances --> if exceeded terminated(all)
- TOTAL NO. OF EC2 INSTANCES IS LIMITED TO = 5
- NOTE: ensure a default vpc exists, creating one if necessary.
